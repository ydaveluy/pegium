# Learn Pegium

The recommended way to learn Pegium is to follow the
[workflow](workflow/index.md).

It takes you from building the repository to grammar design, AST/CST shaping,
cross-reference resolution, validation, formatting, and editor services.

If you prefer learning from a concrete codebase, keep the
[examples overview](../examples/index.md) open while going through the steps.

If you already know the basic workflow and need targeted customization, switch
to [recipes](../recipes/index.md). If you need exact framework concepts, use
[reference](../reference/index.md).
